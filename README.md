# nerdsearch
search nerd font charecters  offline

## instillation

apt update

apt install git 

git clone https://github.com/arNparad0x/nerdsearch

cd nerdsearch

chmod +x nerdsearch

./nerdsearch

## usage

$ `./nerdsearch <search_string>`

add it to your \$PATH to run it without `./`

$`cp nerdsearch /usr/local/bin/`

set an alias

$ `alias ns='nerdsearch'`

to run it like

$ `ns <search_string>`

### example:

>`$ nerdsearch telegram`
